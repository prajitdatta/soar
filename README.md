## 🌈 What is SOAR?

**SOAR** is a bright, fun, motivational browser puzzle game inspired by 2048. Instead of dry numbers, you merge **meaningful milestones** — from a tiny Spark all the way up to Champion — with vibrant candy-colored tiles, bouncy animations, and encouraging messages every step of the way.

The name says it all: **S**tack · **O**vercome · **A**scend · **R**ise.

**[▶ Play it now →](https://prajitdatta.github.io/soar)**

---

## ✨ Features

- **🎨 Vibrant Light UI** — Candy-bright tiles, pastel backgrounds, cheerful animations — no dark mode here!
- **💪 Motivational Theme** — Every tile is a milestone. Every merge is a win. Encouraging messages pop up as you play.
- **🎯 12 Unique Levels** — From Spark to Champion, each tile has a unique emoji, name, and color
- **🎉 Bouncy Animations** — Tiles pop, wiggle, and bounce with personality on every move and merge
- **♻️ Undo Move** — One free undo per move to course-correct
- **🏆 Persistent Best Score** — Your high score saved locally across sessions
- **📱 Fully Responsive** — Swipe on mobile, arrow keys / WASD on desktop
- **⚡ Zero Dependencies** — Pure HTML, CSS, vanilla JS. No frameworks, no build step, no installs

## 🎯 How to Play

1. **Slide** tiles using arrow keys (↑ ↓ ← →) or WASD on desktop
2. **Swipe** on mobile / touch devices
3. **Merge** two matching tiles to evolve them to the next level
4. **Reach** 🏆 Champion (4,096) to win!
5. After reaching 💎 Elite (2,048), keep going to Champion
6. Game ends when no more moves are possible — but every attempt is a step forward!

**Strategy tips:**
- Keep your highest tile locked in a corner
- Build a snake pattern for efficient merges
- Use Undo to recover from a tough spot

## 🚀 Quick Start

### Play instantly

```
https://prajitdatta.github.io/soar-game
```

### Run locally

```bash
git clone https://github.com/prajitdatta/soar.git
cd soar-game
open index.html   # drag into browser — that's it!
```

### Deploy to GitHub Pages

1. Push the repo to GitHub as `soar-game`
2. **Settings → Pages → Source: main branch, / (root)**
3. Live at `https://prajitdatta.github.io/soar` in ~60 seconds

## 🎨 Design Philosophy

SOAR was built to feel **joyful and energetic** from first glance:

- **Color**: Warm cream background, candy-bright tile gradients (coral, amber, sky, mint, violet)
- **Typography**: `Fredoka One` for the chunky playful logo, `Nunito` for readable, friendly UI text
- **Motion**: Springy tile spawn/merge animations, floating score popups, motivation text bursts, wiggling logo icons, animated pastel blob background
- **Tone**: Encouraging — even losing says *"Rise Again!"* and *"Every setback is a setup"*
